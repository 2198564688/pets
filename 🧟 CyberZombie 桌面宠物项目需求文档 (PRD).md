# 🧟 CyberZombie 桌面宠物项目需求文档 (PRD)

## 一、 项目概述

**CyberZombie** 是一款为开发者和极客设计的轻量化 Windows 桌面宠物。它以"赛博丧尸"为核心视觉元素，结合像素艺术风格，通过 Python 后端实现对系统资源的深度监控与剪贴板感知。产品追求极致的交互流畅度，支持高度自定义的动画帧率、显示比例及持久化备忘录功能。

------

## 二、 核心功能需求

### 1. 动力学动画系统

- **状态设计：**
  - **IDLE (待机)：** 5 帧循环。丧尸微动，呼吸感物理回弹。
  - **WALK (行走)：** 6 帧循环。配合水平位移，支持像素级平滑过渡。
  - **PANIC (狂暴)：** 当监测到系统错误（如剪贴板含 `Error`）时触发，红眼闪烁，频率提升。
- **高级控制：**
  - **帧率 (FPS)：** 支持 6-60 FPS 动态调节。
  - **智能翻转：** 运动矢量改变时，图像沿 Y 轴水平翻转。
  - **缩放比例：** 1x - 8x 动态缩放，采用 **Nearest Neighbor** 插值算法确保像素不模糊。

### 2. 交互与物理逻辑

- **拖拽体系：** 鼠标左键点击并拖拽，支持基于惯性的释放效果（Velocity Tracking）。
- **边缘行为：**
  - **碰撞检测：** 碰到屏幕左右边缘自动掉头。
  - **吸附模式：** 右键菜单可开启"自动贴边"，闲置时隐藏至屏幕侧边。
- **点击反馈：**
  - **单击：** 切换 暂停/恢复。
  - **双击：** 快速呼出/隐藏像素备忘录。

### 3. 系统级深度感知 (Cyber Intelligence)

- **实时监控浮窗：** 鼠标悬停 2s 触发，显示透明磨砂质感的像素面板。
  - **CPU/内存：** 实时百分比与进度条。
  - **网络带宽：** 实时上传/下载 KB/s 速率。
- **环境共感：**
  - **剪贴板监听：** Python 后端实时监听。若用户复制了代码报错信息，宠物会做出"吃掉 Bug"的吞噬动画。
  - **进程关联：** 检测到游戏或全屏视频运行时，自动降低自身 CPU 占用并进入低能耗待机。

### 4. 像素备忘录 (Persistence Memo)

- **UI 风格：** 暗黑赛博像素风，白色高对比度文本。
- **核心操作：** 支持新增、删除、排序（新项置顶）。
- **数据持久化：** 自动同步至本地文件（JSON 格式），重启后恢复。
- **时间戳：** 每条笔记自动记录创建时间。

------

## 三、 技术架构与可行性

### 1. 技术栈

- **底层框架：** **Python 3.11+** + **PySide6** (驱动透明异形窗口、系统级 API 调用)。
- **前端渲染：** **React + Framer Motion** (通过 PySide6 的 `QWebEngineView` 嵌入，处理系统级回弹动效)。
- **窗口管理：** 利用 PySide6 的 `Qt.FramelessWindowHint`、`Qt.WindowTransparentForInput`、`Qt.WindowStaysOnTopHint` 及 `setAttribute(Qt.WA_TranslucentBackground)` 实现。
- **系统接口：** 采用 Python 的 `psutil` 库获取硬件状态，`pyperclip` / `QClipboard` 监听剪贴板。

### 2. 性能目标

- **内存占用：** 静态待机 < 60MB，动态监控模式 < 120MB（Python 运行时略有上浮）。
- **渲染：** 稳定 60FPS，无窗口拖动撕裂。

## 四、 交互与右键菜单设计

| **菜单项**   | **子选项/功能**                          |
| ------------ | ---------------------------------------- |
| **状态切换** | IDLE / WALK / 狂暴模式                   |
| **显示尺寸** | 1.0x - 8.0x (步进 0.5)                   |
| **动画速率** | 慢 (10fps) / 标准 (24fps) / 极速 (60fps) |
| **系统看板** | 强制显示 / 自动隐藏                      |
| **数据管理** | 导出备忘录 / 清除缓存                    |
| **退出**     | 优雅关闭所有 Python 线程并保存状态         |

## 六、 未来扩展方向

- **义体进化：** 互动次数增加后，像素形象增加机械臂或电子眼。
- **本地 AI 接入：** 备忘录集成简单的本地 LLM 总结功能。
